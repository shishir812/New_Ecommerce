# E-Commerce Frontend

React and Vite frontend for the full-stack e-commerce platform. It provides a responsive customer storefront and an admin dashboard connected to the Django REST API.

## Features

- Product listing, category filtering, search, product details, and review display.
- Cart, checkout, customer login, registration, account profile, and order history.
- Admin views for products, categories, orders, customers, low-stock items, and reviews.
- Ant Design components for a polished and consistent UI.

## Setup

```powershell
npm install
npm run dev
```

The frontend expects the backend API to run at `http://127.0.0.1:8000/api`.
